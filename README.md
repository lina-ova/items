# Product Inventory Monorepo

A full-stack product management application with a **FastAPI backend** and a **React frontend**.  
The backend provides an API for managing products, and the frontend is a user-friendly React app to interact with the API.

---

```text
├─ frontend/      # React frontend
│   ├─ src/

│   └─ ...
└─ README.md
```
